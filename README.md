## 安裝套件
  * node v16.15.1
  * Express v4.17.1


## 執行專案
1. 將本專案 clone 到本地端

2. 安裝套件
```shell
  npm i express@4.17.1
  npm i mongoose@6.0.4 # Mac M1 chip
```
3. 啟動伺服器
```shell
  npm run start
```
4. 終端機啟動成功會顯示以下訊息
```node
  App is running on http://localhost:3000
```
